1)Download/Install XAMPP : https://www.apachefriends.org/xampp-files/5.6.30/xampp-win32-5.6.30-0-VC11-installer.exe
2) Go to https://pecl.php.net/package/mongodb/1.2.5/windows
3) Download "5.6 Thread Safe (TS) x86"
4) Copy the "php_mongodb.dll" file in to "C:\xampp\php\ext"
5) Add "extension=php_mongodb.dll" to the end of the "C:\xampp\php\php.ini" file
6) Install MongoDB
